# SOLAR SDG

## 專案架構

```
solar-sdg/
│
├── app/
│   ├── __init__.py
│   ├── routes.py
│   ├── models.py
│   └── templates/
│       └── index.html
│
├── static/
│   └── (靜態檔案如 CSS, JS, 圖片)
│
├── config.py
├── run.py
└── README.md
```

- `app/`：Flask 應用程式主目錄
- `static/`：靜態檔案
- `config.py`：設定檔
- `run.py`：啟動 Flask 應用程式
